Frequently asked questions

Have questions? We are here to help

What is the FAQ component?

The FAQ component is an accordion style component that displays question and answer pairs. The initial state of the question is closed, and when the user clicks on the question, the answer is revealed.

How does the component work?

The component works by using JavaScript to toggle the visibility of the answer when the user clicks on the question. The component is designed to be accessible and responsive to different screen sizes.

Is the component accessible?

Yes, the component is designed to be accessible and meet the accessibility guidelines.
